# 📊 Amazon 1M Records Sales Analysis | Excel

![Dashboard Preview](dashboard-preview.png)

## 🎯 Problem Statement
Analyzed 1M+ Amazon sales records to identify return patterns, optimize delivery methods, and reduce revenue loss.

## 🛠️ Tools Used
Microsoft Excel, Power Query, Pivot Tables, Advanced Charts, Dashboards

## 📌 Key Insights
1. **Cash on Delivery has 35% return rate** vs 12% for Prepaid orders
2. **Top 5 States** contribute 60% of total sales but also 45% of returns
3. **Electronics category** shows highest revenue but also highest return %

## 📈 Business Impact
Proposed strategy to promote prepaid orders in high-return states, potentially saving 20% revenue loss.
